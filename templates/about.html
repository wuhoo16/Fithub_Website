About Us

In today's world where desk jobs are prevalent, over 60-85% of people around the world lead sedentary lifestyles. This lack of physical activity can have serious implications for overall health and is attributed to about 2 million deaths every year. In fact, the World Health Organization states that "sedentary lifestyles increases all causes of mortality, double the risk of cardiovascular diseases, diabetes, and obesity, and increase the risks of colon cancer, high blood pressure, osteoporosis, lipid disorders, depression and anxiety."


Here at Fithub, we believe that everyone around the world should have the resources and opportunity to improve their fitness and strive for a more active lifestyle. We are aware of the dangers of a sedentary lifestyle and want to provide detailed information to easily allow our users to incorporate regular exercise into their lives.


However, we are aware that it can be difficult to find the information you need with so much scattered information over the web. In addition, we know that the ongoing COVID-19 pandemic may have limited your access to public gyms and imposed major challenges to finding and sticking to a workout routine. As a result, we have aggregated a database of over 100 exercises and provided links to common workout equipment and popular fitness Youtubers to help your overcome these challenges. With our database, you can easily:

  • Filter exercises by category or target muscle groups to easily build your own tailored workout routine
  • Read detailed guides about how to perform specific exercises and easily navigate to instructional videos
  • Check the equipment needed to perform certain exercises at home
  • Check the price of various fitness equipment/gear
  • Check the rating of the vendors selling fitness equipment/gear
  • Find discounts on fitness products with links to Youtube influencers with sponsors
  • Find deals on fitness products using links to Youtube channels with related sponsorships
  • Learn more about specific exercises and equipment referenced in fitness Youtube videos

Athletes and fitness enthusiasts of all ages and experience levels are welcome to explore our comprehensive database about specific exercises, equipment, and Youtubers!

Additional Information

Data and Links:

  • Youtube Data V3 API Guide

    This API was used to setup and activate the Youtube API. It includes supported API operations and syntax examples to get different details about channels related to fitness. Channels, search, playlists, and video API operations were used.

  • wger REST API

    This API was used to grab data on 700+ exercises with information like exercise category, muscle groups, equipment, images, comments, etc. Out of the 700+ exercises, only 110 exercises were used since they were approved, in english, and had information for name, description, category, and equipment. The request made to the API was parsed into a JSON format. Then, the 110 exercises were filtered out along with specific information for each exercise. Since over half of the exercises didn't come with images, the Google Custom Search JSON API was used to search for images related to each exercise.

  • Google Custom Search JSON API

    This API was used to do a Google image search. We sent an HTTP GET request to the URI and passed in specific parameters like image file type, API key, search engine key, and search string. The call returns a response which contains a Search instance. The Search instance contains information from the search result which we used to grab the image URLs. The image URLs were saved corresponding to the exercise.

  • SERPSTACK API Documentation

    SERPSTACK was the API we used for finding results for fitness equipment. First, we did a pull to get an example JSON file of the search results, then used the information in the JSON file to construct a method for making a list of Equipment objects. We then used this list in the Equipment page to list search results in a grid.
